MySQL Table Schema

Please ensure your MySQL database has the required database and table structure configured beforehand to support uploads and query parameter inserts properly.

SQL Statement 
CREATE DATABASE IF NOT EXISTS school_db;
USE school_db;

CREATE TABLE IF NOT EXISTS students (
  sl INT AUTO_INCREMENT PRIMARY KEY,
  photo VARCHAR(255) DEFAULT NULL,
  roll VARCHAR(50) NOT NULL,
  name VARCHAR(255) NOT NULL,
  class VARCHAR(50) NOT NULL,
  section VARCHAR(50) NOT NULL,
  guardian VARCHAR(255) DEFAULT NULL,
  phone VARCHAR(50) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
Primary Key 'sl' handles autoincrement sequence numbers natively.

Deploy Environment Steps

  • 1 Copy db.php, insert.php and student_form.html to your PHP workspace.
  • 2 Configure db host, name, and passwords inside the top section of db.php.
  • 3 Ensure your server has write permissions on the directory to auto-create the ./uploads/ folder.
